## Postman Collection — Inventory API Lab 6

 Колекція містить 8 запитів:
- Register item (`POST /register`)
- Get all inventory (`GET /inventory`)
- Get item by ID (`GET /inventory/:id`)
- Get item photo (`GET /inventory/:id/photo`)
- Update item (`PUT /inventory/:id`)
- Update item photo (`PUT /inventory/:id/photo`)
- Delete item (`DELETE /inventory/:id`)
- Search item (`POST /search`)

 Експортовано у форматі `Collection v2.1`  
 Файл: `InventoryAPI.json`  
 [Колекція в Postman Workspace](https://stepangavriliv73-3752676.postman.co/workspace/...)

 Запити протестовані локально на `http://localhost:3000`
